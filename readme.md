
This is a simple example of  Webklex/laravel-imap,  Here we are trying to get the data from a mail folder . In this example we are using gmail and laravel verion 5.6.

 # *Inbox
 # *Sent mail
 # *Important
 # *Spam
 # *Starred
 # *Trash
 # *Drafts
